Contrast

Moodle's use of colour in the Boost and classic themes and the impact it has on accessibility

Colour contrast checks

The colour contrast of the components on this page are checked with some JS that adds a badge like this: 2.50:1 if a the colour contrast of the component does not meet the WCAG standards. For AA compliance with these standards the number should be at least 4.5:1 to pass.
